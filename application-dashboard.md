<p><h2>Tasks Panel,</h2></p>

<p>This allows you to define background tasks that Burp Suite will perform as you use the application.<br>
The default task is "Live Passive Crawl" which automatically logs the pages visited.</p>

<p><h2>Even Log,</h2></p>

<p>Will outline the actions performed by Burp Suite such as starting the proxy and connection details.</p>

<p><h2>Issue Activity,</h2></p>

<p>Exclusive to Burp Suite Professional, it displays the vulnerabilities identified by an automated scanner.<br>
It ranks the vulnerabilities by severity and certainty.</p>

<p><h2>Advisory,</h2></p>

<p>Provides details about identifed vulnerabilities, including references and suggested remediations.<br>
These details can aso be exported into a report, community edition may not show vulnerabilities.</p>
