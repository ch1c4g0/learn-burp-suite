

<p>The target tab provides multi functionality such as, mapping sites, issue definitions, and scope settings.</p>

<p><h1>Site Map</h1></p>

<p>This sub tab allows us t map web applications in a tree like structure.<br>
Every page we visit while the proxy is active will be displayed on the site map.<br>

This feature enables us to build a better picture of the site while simply browsing it.<br>

It is particularly useful to map out API's.</p>

<p><h1>Issue Definitions</h1></p>

<p>Although this is mainly a feature for burp suite professional, we can utilize the vulnerability list that the premium scanner looks for.<br>
This can serve as a valuable reference for reports or describing vulnerabilites that may have been identified.</p>

<p><h1>Scope Settings</h1></p>

<p>This allows us to control the target scope, we can include or exclude IPs/Domains, allowing us to focus soley on the web applications we are testing.</p>
