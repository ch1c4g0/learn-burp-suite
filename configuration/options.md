
<p>When configuring Burp Suite there are two types of settings, Global and Project.</p>

<p><h2>Global Settings</h2></p>

<p>Affect the entire Burp Suite installation and are applied every time you start the application.<br>
Think of it as a baseline configuration for Burp Suite.</p>

<p><h2>Project Settings</h2></p>

<p>Settings specific to the current project, they apply only when in a saved project.<br>
It's important to note, community edition does not allow saving of projects, when you close the application all data is lost.</p>

**<p>Use the search feature for settings, it'll save you a lot of time.</p>**
